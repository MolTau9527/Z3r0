# 社区

Z3r0 社区面向关注授权安全评估工作流的用户、安全研究人员和软件工程师。

## 讨论范围

社区讨论与贡献主要围绕以下方向：

1. 多 Agent 运行时设计、上下文管理与任务连续性。
2. 安全评估工作流、代码审计、漏洞验证与证据模型。
3. sandbox 隔离、分布式执行资源、受控 egress 与运行安全。
4. LightRAG 文档接入、检索质量、图谱数据与 model 集成。
5. 前端可用性、API 契约、文档、测试与部署可靠性。

## 参与规范

技术讨论应保持具体、可验证和相互尊重。涉及安全敏感问题时，请遵循负责任披露原则，不公开凭据、私有目标数据或任何超出授权范围取得的材料。

Z3r0 的使用必须遵守适用法律以及系统所有者定义的授权边界。项目仅用于授权测试、风险评估、代码审计和安全研究。

## 参与贡献

可通过 GitHub Issues 提交可复现的问题报告和范围明确的功能建议。Pull Request 应保持改动聚焦，说明用户可见行为，提供与风险相匹配的验证，并在契约或工作流发生变化时同步更新文档。
